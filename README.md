# Running Locally
 Clone the Repo

**Frontend**
1. `cd 3D-Maps/`
2. Add .env file with varibles MAPTOKEN and BASEURL
3. `npm install`
4. `npm start`

**Backend**
1. `cd backend/`
2. 2. Add .env file with varible ATLAS_URI
3. `npm install`
4. `nodemon server.js`

Tech Stack Used:
1. MongoDB
2. Express JS
3. React JS
4. Node JS
5. Babylon JS
6. MapBox
